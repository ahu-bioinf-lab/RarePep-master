# RarePep-master
This repository contains the code for our paper: RarePep: A Cross-Pathogen Transfer Learning Framework for Peptide Prediction of Specific Parasites

## Experiments

### 1) Transductive Experiment
The transductive experiment is located in RarePep/transductive/ceshi1_main_XX.py.
To run:
python ceshi1_main_XX.py

### 2) Transfer Experiment
The transductive experiment is located in RarePep/transfer/transfer_main_parasite_finetuneXXX.py.
To run:
python transfer_main_parasite_finetuneXXX.py

### 3) Cross-Domain Experiment
The transductive experiment is located in RarePep/CDAN-DANN/CDAN-DA-XXX-ceshi.py.
To run:
python CDAN-DA-XXX-ceshi.py
